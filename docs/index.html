What is .npy? ▼

.npy is NumPy's native single-array binary format. It stores the array data type, shape, and raw data in a compact binary file with a short ASCII header. It is the most efficient way to save and load numerical arrays in Python.

Load in Python:

import numpy as np
sigma = np.load("compton_sigma_T053_16g_10a.npy")
print(sigma.shape)   # (16, 16, 10)
# sigma[g_in, g_out, angle_bin]

Array axes:

  • Axis 0 — Incoming photon energy group
  • Axis 1 — Outgoing photon energy group
  • Axis 2 — Scattering angle bin (ξ = cos θ)
The Multigroup Formula ▼

The multigroup, angle-binned scattering cross section is defined as:

σ(g→g', [ξi,ξi+1]; T) = 2π ∫ w(E)·ΣE(E,E',ξ) dξ dE' dE ⁄ ∫ w(E) dE

With uniform weight w(E) = 1. Collapsing to coarser groups uses the same weight: incoming groups are width-averaged (wg = Eg+1 − Eg), outgoing groups and angle bins are summed.
